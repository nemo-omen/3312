Brainstorms

I'd like to create a Pomodoro timer app that allows users to create, start, and stop Pomodoro timers. These timers run in cycles of 25 minutes and 5 minutes, for work time and break time respectively. The user should be able to adjust the work time and break time, start and stop the timers, and save specific timer configurations.
