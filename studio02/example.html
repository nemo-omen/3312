Elements everywhere

Different HTML elements have different semantics (inherent meanings) and different default appearances.  See the comments in this HTML document for more information.

Block-level elements

Most HTML elements are block-level—they have their own block-shaped area that can stretch across the document.  Many block-level elements are meant to be used in particular ways, to provide meaning and structure to the document.

HTML character entities

Some characters, such as < and >, have special meanings in HTML and so must be escaped.  The following table gives a few important ones.

entity result meaning
&lt; < less-than symbol
&gt; > greater-than symbol
&quot; " quotation mark
&amp; & ampersand symbol
&nbsp;   non-breaking space
